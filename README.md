SpeedtestStatistics
===================
Uses speedtest_cli: https://github.com/sivel/speedtest-cli (uses apache license)

# Prerequisities:

* Perl
* Python
* bash
* R programming language


# Installation:

Checkout Repo.
run speedtest_start (generates result-file and image)
optional: create cron-job
